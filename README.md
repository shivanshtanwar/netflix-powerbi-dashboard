## 🎬 Netflix Global Content Analysis Dashboard (Power BI)

## 📊 Project Overview
In this project, I built a Power BI dashboard to analyze Netflix content across countries, genres and trends.
The goal is to uncover insights into content trends, production patterns, and regional dominance.
This project is part of my data analytics portfolio to showcase my Power BI and data storytelling skills.

## 🎯 Objectives
Analyze global content distribution.
Identify top content-producing countries (e.g. United States, India)
Compare Movies vs TV Shows.
Explore genre popularity and trends.
Understand content growth over time.

---

## 🧹 Data Cleaning & Transformation
Data preprocessing was performed using Excel and Power Query:

- Split multi-country entries into row-level format  
- Handled missing/null values in the country column  
- Standardized inconsistent country names  
- Removed duplicates and unnecessary columns  
- Created a separate mapping table for accurate geographic analysis
  
---

## 📈 Key Insights
- The United States and India contribute the highest number of titles in the dataset  
- Movies significantly outnumber TV Shows  
- Content production increased rapidly after 2015  
- Drama and International content are among the most dominant genres
